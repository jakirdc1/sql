CHAPTER SEVEN

Demonstrate Understanding Of The Main Features Of Structured Programming

106. Argument Table

Argument table is a table that is searched.

107. Array

Array is a collection of items of the same type and size of data. It's alternative name is table.

108. Binary Search

Binary search is a technique for searching an ordered argument table that is efficient for large tables.

109. Direct Table Addressing

Direct table is a technique for accessing a function table (without searching an argument table) by deriving the position in the function table directly from the search argument.

110. Discrete Table

Discrete table is an argument table in which each entry represents a particular value that will be compared with a search argument in an attempt to find an exact match.

111. Function Table

Function table is a table that contains values that are to be retrieved for use in processing.

112. Index

Index is used in conjunction with a table name to specify a particular element of a table; a part of an indexed file.

113. Multidimensional Table

Multidimensional table is a table that can be visualized as containing more than a single column. A two-dimensional table contains rows and columns; a three-dimensional table is a stack of two-dimensional tables.

114. One-Dimensional Table

One-dimensional table is a table that can be visualized as containing a single column of values.

115. Paired Table

Paired table is a table where argument and function table used together. When the proper entry in the argument table is found, the corresponding element of the function table is retrieved.

116. Search Argument

Search argument is the value that is compared with argument table entries.

117. Segment Table

Segmented table is an argument table in which the argument entry is the upper (for an ascending table) or lower (for a descending table) limit of a range of values.

118. Sequential Search

Sequential search is a method for searching an argument table that examines the entries in the order in which they appear in the table, starting with the first entry.

119. Single Table

Single table is an argument table with no corresponding function table (used in editing data) or a function table with no corresponding argument table (used in direct table addressing).

120. Subscript

Subscript and index are same. So index is used in conjunction with a table name to specify a particular element of a table; a part of an indexed file.

121. Table

Table is a collection of items of the same type and size of data. It's alternative name is array.

122. Table File

Table file is a file in which the data that makes up a table is stored.

123. Table Lookup

Table lookup is the examination of an argument table to find an entry that is equal to the search argument (in the case of discrete table), greater than or equal to the search argument (in the case of segmented table in ascending order), or less than or equal to the search argument (in the case of segmented table in descending order).
